Projects

Meditate Now

Meditate Now provides guided meditations based on emotion tags and was inspired by my friend who is a cancer survior and wanted meditations to help deal with the emotions she was feeling after her diagnosis. Meditate Now was created using MongoDB, AngularJS, Express.js, and Node.js. The mp3 files are hosted on AWS S3.

ReadMe

ReadMe is a Ruby on Rails app that digitizes the bookclub exprerience. The user is able to start a bookclub, join a bookclub, search for bookclubs, as well as mark books as favorites, to-read, or have read. There is also comment functionality for the bookclub threads. The app also utilizes the Google Books api, SASS, OAuth, and Cloudinary.

Flixr

Flixr is a social app that lets the user favorite streaming shows and show their current watch list to friends. The user can search for shows and see on which platforms the shows are streaming. Flixr was created using Node.js, Express.js, Passport.js, Bcrypt.js, Materialize, Guidebox api, and Cheerio.js.

Particle Game

This is a browser-based game employing javascript canvas to draw the circlular bomb elements and the randomly-generated particles. The object is to place a bomb on the screen and take out as many of the particles as possible. The game ends when the bomb radius reaches a maximum value.
