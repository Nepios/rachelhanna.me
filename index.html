Projects

Meditate Now

Meditate Now is a meditation app that provides a curated list of guided meditations that are playable within the app. The user selects a current emotion and then populates a list of meditations that are relevant to that emotion. If the user just wants to meditate on his/her own, a countdown timer is provided that allows the user to add time in 5 minute increments.

Meditate Now was created using MongoDB, AngularJS, Express.js, and Node.js. The mp3 files are hosted on AWS s3 as many of them exceeded the size limit limit for storage in MongoDB.

ReadMe

ReadMe is a Ruby on Rails app that digitizes the bookclub exprerience. The app also utilizes the Google Books api, SASS, Oauth, and Cloudinary.

Flixr

Flixr is a social app that lets the user favorite streaming shows and show their current watch list to friends. The user can search for shows and see on which platforms the shows are streaming.

Flixr was created using Node.js, Express.js, Passport.js, Bcrypt.js, Materialize, Guidebox api, and Cheerio.js.

Particle

This is a browser-based game employing javascript canvas to draw the circlular bomb elements and the randomly-generated particles. The object is to place a bomb on the screen and take out as many of the particles as possible. The game ends when the bomb radius reaches a maximum value.
